## 🎨 Color Scheme

### Primary Colors
```
🔵 Blue:     #2563EB  (Primary buttons, highlights)
🟣 Purple:   #9333EA  (Accents, gradients)
🟢 Green:    #16A34A  (Success states, checkmarks)
🟠 Orange:   #EA580C  (Warnings)
🔴 Red:      #DC2626  (Errors)
```

### Gradients
```
Hero:        from-blue-600 to-purple-600
Cards:       from-blue-50 via-white to-purple-50
Buttons:     from-blue-600 to-purple-600
Guest Banner: from-purple-100 to-blue-100
```

---

## 🎭 Interactive States

### Form Fields
```
Default:     Border gray-200, bg-white
Focus:       Border blue-500, ring-4 ring-blue-100
Error:       Border red-500, text-red-600
Success:     Border green-500, checkmark visible
```

### Buttons
```
Default:     bg-gradient, shadow-lg
Hover:       shadow-xl, scale-105
Active:      scale-95
Disabled:    opacity-50, cursor-not-allowed
Loading:     Spinner animation
```

### Quick Amount Buttons
```
Default:     bg-gray-50, border-gray-200
Hover:       bg-gray-100
Selected:    bg-gradient blue-to-purple, text-white
```

---

## 📐 Layout Specifications

### Mobile (< 768px)
```
Container:    Full width, px-4
Cards:        Single column, full width
Buttons:      Full width, py-4
Typography:   Scaled down (text-2xl → text-xl)
Features:     Stacked vertically
```

### Tablet (768px - 1024px)
```
Container:    max-w-4xl, px-6
Cards:        2 columns where applicable
Buttons:      Flexible width
Typography:   Standard size
Features:     2x3 grid
```

### Desktop (> 1024px)
```
Container:    max-w-7xl, px-8
Cards:        3 columns
Buttons:      Auto width, centered
Typography:   Full size (text-5xl)
Features:     3x2 grid
```

## 🎬 Animations

### Page Load
```css
Fade in:     0.3s ease-out
Slide up:    0.4s ease-out
Cards:       Stagger 0.1s each
```

### User Interactions
```css
Button click:   scale(0.95) 50ms
Button hover:   scale(1.05) 200ms
Checkmark:      fade + scale 300ms
Error shake:    translateX(±10px) 200ms
```

### Loading States
```css
Spinner:     rotate 1s linear infinite
Progress:    width transition 300ms
Skeleton:    pulse animation
```

## ✅ Accessibility

### Keyboard Navigation
```
Tab Order:
1. Back button
2. Email input
3. DOB input
4. Password input
5. Confirm password input
6. Submit button
7. Sign in link
```

### Screen Readers
```
Labels:      All inputs have labels
ARIA:        Proper role and aria-labels
Alt Text:    All icons described
Focus:       Visible focus ring
```

### Color Contrast
```
Text:        WCAG AA compliant
Links:       Underlined on hover
Errors:      Color + icon + text
Success:     Multiple indicators
```
