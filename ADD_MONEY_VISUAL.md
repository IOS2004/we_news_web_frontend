## 🎨 Color Palette

### Primary Colors
```
🟠 Primary Orange:    #FF6B35  (Buttons, highlights)
⚫ Text Primary:      #2C3E50  (Headings)
⚪ Background:        #F5F7FA  (Page background)
⬜ Card Background:   #FFFFFF  (Cards)
```

### Accent Colors
```
🔵 Info Blue:         #3498DB  (Security notice)
🟢 Success Green:     #27AE60  (Checkmarks)
🔴 Error Red:         #E74C3C  (Validation errors)
🟡 Warning Yellow:    #F39C12  (Warnings)
```

### Gradient
```
Wallet Balance Card:
  from-primary (#FF6B35)
  to-primary-dark (#E65428)
  Linear gradient, 45° angle
```


## 🎭 Interactive States

### Quick Amount Buttons
```
Default:   border-gray-200, hover:border-primary
Selected:  border-primary, bg-primary, text-white
Hover:     transform scale(1.02), shadow-md
```

### Payment Method Cards
```
Default:   border-gray-200, bg-white
Selected:  border-primary, bg-primary/5, checkmark
Hover:     border-gray-300, shadow-sm
```

### Submit Button
```
Default:   bg-primary, text-white
Hover:     bg-primary-dark, shadow-lg
Loading:   opacity-60, spinner icon
Disabled:  opacity-40, cursor-not-allowed
```

## 🎬 Animations

### Page Load
```
Fade in from top: 0.3s ease-out
Cards stagger:    0.1s delay each
```

### Button Click
```
Scale down:  0.95 (50ms)
Scale up:    1.0 (100ms)
Ripple:      Radial gradient (300ms)
```

### Success Flow
```
1. Button loading state (spinner)
2. Success toast (slide in from top)
3. Wallet refresh (fade transition)
4. Navigate to wallet (slide right)
```

## 🔢 Validation Feedback

### Error States
```
Input Border:    border-red-500
Error Text:      text-red-600, text-sm
Icon:            ⚠️ (Inline with message)
Animation:       Shake (0.2s)
```

### Success States
```
Toast:           Green background, white text
Icon:            ✓ Checkmark
Position:        Top-right corner
Duration:        3 seconds
```

## 📊 Responsive Breakpoints

```css
/* Mobile */
< 768px:  Single column, full width
          Stack all cards vertically
          Reduce padding to 1rem

/* Tablet */
768px - 1024px:  2 columns start here
                 Sidebar moves to right
                 Maintain spacing

/* Desktop */
> 1024px:  Full 2-column layout
           Max width container
           Optimal spacing
```

## 🎯 Focus States

### Keyboard Navigation
```
Tab Order:
1. Back button
2. Amount input
3. Quick amount buttons (left to right)
4. Payment method cards (top to bottom)
5. Submit button
6. Sidebar buttons

Focus Ring:
- 2px solid primary color
- 2px offset
- Rounded corners
```
